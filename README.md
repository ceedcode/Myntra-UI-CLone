## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/ceedcode/Myntra-UI-CLone.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Myntra-UI-CLone
    ```

3. Open `index.html` in your web browser:
    ```bash
    open index.html
    ```
